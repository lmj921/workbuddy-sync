# 方案：根据「原始组织」列填充「工种」列

## 文件
`/Users/a1_builder/Downloads/北极光A1工作室4-5月token使用情况.xlsx`，2个Sheet（Sheet1=4月，Sheet2=5月）

## 现状
- **工种列**：全部为空（NaN）
- **原始组织列**：包含完整组织路径，可提取末尾组名推导工种
- Sheet1 有 87 行空白行（全 NaN），Sheet2 无空白行

## 映射规则

| 原始组织关键词 | 工种 |
|---|---|
| `/运营组` | 运营 |
| `/美术组` | 美术 |
| `/客户端组` 或 `/客户端一组` 或 `/客户端二组` | 客户端 |
| `/引擎组` | 引擎 |
| `/服务器组` | 服务器 |
| `/策划组` 或 `/策划一组` 或 `/策划二组` 或 `/策划三组` | 策划 |
| `/PM组` | PM |

## 待确认的特殊情况

### 1. vincentren（两个Sheet各1行）
- 原始组织：`IEG.../北极光A1工作室`（到工作室级别就结束了，无子组）
- 职族：中干及以上
- **确认**：工种填「管理」

### 2. 腾娱子公司员工（Sheet1约19人，Sheet2约19人）
- 原始组织统一为 `子公司组织/腾娱/腾娱A研发部/A1业务中心`，无组别细分

1. 用 openpyxl 加载 Excel，遍历两个 Sheet

3. 腾娱子公司按职族映射：T族→技术，P族→策划，D族→美术
4. vincentren 工种填「管理」
5. 保存到原文件
