# 三大渠道风格索引

> 资深财经编辑技能目前收录 3 种渠道风格，对应 3 套独立的规则文档。
> Agent 在收到写作任务时，先根据用户指定的 `channel` 参数加载对应目录的规则包。

---

## 一、三风速查

| 风格代号 | 中文名 | 目标读者 | 首要目标 | 语气 | 结尾方式 | 典型场景 |
|---|---|---|---|---|---|---|
| `internal` | **严肃财经风** | 机构、从业者、监管可见 | 准确、权威、合规 | 零情绪、客观冷静 | 数据 / 展望收 | 券商日报、机构研报、监管合规内容 |
| `wechat` | **公众号风** | 订阅用户、准专业散户 | 共鸣、陪伴、留存 | 朋友聊天、轻度态度 | 三件套（窗口+互动+声明） | 公众号日更、解读稿、周度复盘 |
| `toutiao` | **头条吸睛风** | 广域散户、路人流量 | 点击、完读、互动 | 强钩子、冲突化、故事化 | 追问 / 清单 / 金句收 | 头条号、百家号、算法分发信息流 |

## 二、三风的 DNA 差异

| 维度 | internal | wechat | toutiao |
|---|---|---|---|
| 标题范式 | 主体 + 动作 + 量化 | 钩子 + 金句 + 画面 | **数字 + 冲突 + 追问** |
| 标题字数 | 15–25 字 | 10–20 字 | **18–40 字** |
| 标题情绪 | 零 | 轻度 | **中度偏高** |
| 感叹号 | 禁用 | ≤ 1 | **允许 1–2** |
| 首段要求 | 6 要素完整 | 3 行法破题 | **反常识 + 数字 + 反转** |
| 段长 | 无硬性上限 | ≤ 3 行 | **≤ 3 行，多用单句成段** |
| 数字密度 | 高（精确） | 中 | **极高（冲击）** |
| 比喻允许度 | 零 | 生活化 | **生活化 + 痛点化** |
| 第二人称 | 禁用 | 可用"咱们" | **可用"你""散户"** |
| 追问 / 反问 | 禁用 | 可用（机制解读） | **强制 ≥ 1 处（情绪型）** |
| 互动引导 | 禁用 | 三件套必备 | **禁用**（算法不吃） |
| emoji | 禁用 | 引导处 👇 | **禁用** |
| 合规声明 | 不需要 | 长版 | **短版一句话** |
| 风险打压 | 几乎没有 | 轻 | **高**（易因标题党被限流）|

## 三、同一事件的三风改写示例

**原始事实**：锋龙股份连续收获第 17 个涨停板，动态市盈率达 4700 倍，市值 220 亿元，2025 年全年净利润 3000 万元。

### internal 版本
> **锋龙股份再收一字板：连续 17 个交易日涨停，动态市盈率 4700 倍**
>
> 截至 4 月 23 日收盘，锋龙股份（002931.SZ）连续收获第 17 个涨停板，股价报 47.38 元，总市值 220 亿元。按 2025 年归母净利润 3000 万元测算，公司动态市盈率达 4700 倍。公司主营园林机械零部件，近期因"人形机器人概念"受到市场追捧。同期深证成指上涨 0.8%，中证 1000 指数上涨 1.2%。

### wechat 版本
> **17 连板的锋龙股份，到底是谁在买？**
>
> 这家做园林机械零部件的小公司，最近 17 天一字板没停过。
>
> 市值从 13 亿冲到 220 亿。4700 倍市盈率。一年挣 3000 万。
>
> 啥概念？相当于这家公司现在就是安安静静躺平不涨了，需要 4700 年才能靠净利润把你的买入成本赚回来。
>
> 游资说这是"人形机器人"。但订单在哪？业绩在哪？都还在 PPT 里。
>
> 一句话总结：烟花很美，烟花也很快会灭。这种妖股看个热闹就行，真上车得先想清楚——最后一棒会不会是自己。
>
> 风险提示：以上不构成投资建议。

### toutiao 版本
> **《17 连板！4700 倍市盈率：锋龙股份的资本赌局，散户接盘是福是祸？》**
>
> 27 天 7 板，再 17 连板，市值飙至 220 亿！
>
> 4700 倍市盈率的狂欢背后，是业绩仅 3000 万的残酷现实。
>
> 游资把"机器人概念"炒到天上，股价脱离地心引力。
>
> 当 4700 倍市盈率遇上 3000 万净利润，这已经不是估值，这是玄学。
>
> 多头说"机器人产业方兴未艾"；空头说"17 连板之后能涨到月球吗"。
>
> 但烟花再美也会灭：当借壳预期落空、概念退潮，最后一棒谁来接？
>
> 这场资本狂欢，注定是**少数人的盛宴、多数人的坟场**。
>
> 以上内容不构成投资建议，仅供参考。

## 五、调用协议（供上层 Agent 参考）

```
输入参数：
  - content_form: 快讯 / 盘面点评 / 一文读懂 / 专题 / 滚动播报 / ...
  - channel: internal / wechat / toutiao
  - source_material: 原始素材（公告 / 数据 / 事实 / 另一风格稿件）

Agent 加载顺序：
  1. _common/compliance-checklist.md          # 通用合规（全渠道必读）
  2. _common/title-rules.md                   # 通用标题规则
  3. _common/lead-paragraph.md                # 通用导言规则
  4. _common/editorial-taboos.md              # 通用禁忌
  5. _common/quality-rubric.md                # 通用质量评分
  6. _channels/{channel}/style.md             # 指定渠道 DNA
  7. _channels/{channel}/title-patterns.md    # 指定渠道标题
  8. _channels/{channel}/content-patterns.md  # 指定渠道正文
  9. _channels/{channel}/antipatterns.md      # 指定渠道反例（如有）

产出顺序：
  1. 标题（3 选 1）
  2. 正文
  3. 自检清单（对照本渠道的 rubric）
```
