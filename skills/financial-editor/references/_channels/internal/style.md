# 严肃风（internal）— 风格 DNA

> 适用渠道：腾讯网财经、证券 App、官网、官方研究报告前置。
> 参考语料：腾讯网编辑手册 + 证券产品部内容中心编辑手册。

## 一、三度标准（Speed × Depth × Edge）

| 维度 | 要求 | 自检 |
|---|---|---|
| **速度** | 突发 T+5 分钟内快讯；重大事件 T+30 分钟解读；数据发布 T+60 分钟数据稿 | 稿件时间戳 vs 事件时间戳 ≤ SLA |
| **厚度** | 至少 2 个归因维度、2 个以上信源、1 个前瞻判断 | 对照 quality-rubric 第 3、5 条 |
| **锐度** | 有主干观点、不和稀泥、有取舍 | 读一段能复述观点 = 达标 |

## 二、9 条风格原则

1. **事实优先**：先陈述，再解读，不倒置。
2. **数据本位**：关键论断必须有数字支撑，不讲"大概""差不多"。
3. **克制修辞**：不用夸张形容词（血洗/暴击/炸裂/崩了）。
4. **判断鲜明**：有观点不含糊，"两边都有道理"只能作为背景。
5. **源必注明**：任何引述必带"谁在何时何处说了什么"。
6. **读者分层**：专业术语首次出现必配白话解释，给非金融专业用户兜底。
7. **平衡客观**：涉公司/个股必有多方视角，结尾加合规声明。
8. **结构节制**：小标题清晰，段不过 4 行，屏不过 1 图。
9. **向前看**：每篇都要有"接下来看什么"的观察窗口。

## 三、风格 DNA 参数表

| 参数 | 取值 |
|---|---|
| 人称 | 第三人称为主，避免"我们""小编" |
| 语气 | 陈述 > 抒情；克制 > 渲染 |
| 句长 | 平均 20–35 字；长句必拆 |
| 段长 | ≤ 4 行（移动端） |
| 形容词密度 | 每 200 字 ≤ 3 个 |
| 数字密度 | 每 300 字 ≥ 2 个 |
| 术语密度 | 每 500 字 ≤ 5 个（首次必解释） |
| 感叹号 | 0（严禁） |
| 表情符号 | 0（严禁） |
| 网络热词 | 0（严禁） |
| 引语来源 | 100% 可核实 |

## 四、严肃风 vs 其他风格对照

| 场景 | 严肃风（internal） | 公众号（wechat） | 头条（toutiao） |
|---|---|---|---|
| 标题风格 | 陈述+数据 | 人格化+钩子 | 冲突+数字 |
| 用词 | 规范书面 | 可适度口语 | 口语+冲突词 |
| 段落 | 紧凑节制 | 可抒情断行 | 短句短段 |
| 配图 | 数据/现场为主 | 封面+品牌水印 | 强视觉冲击 |
| 结尾 | 前瞻+合规声明 | 互动引导+合规 | 悬念/争议+合规 |

## 五、禁止事项（严肃风专属）

- 禁止第一人称"我""我们"（除署名评论外）
- 禁止网络梗、缩略表情（yyds、绝绝子、栓 Q 等）
- 禁止段首空两格以外的异形排版
- 禁止标题使用多个标点、感叹号、问号嵌套
- 禁止正文中出现"点击上方关注""转发朋友圈"等公众号话术
