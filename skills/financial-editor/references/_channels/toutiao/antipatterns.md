# 头条吸睛风·反例对比手册

> **头条风最危险的地方：钩子稍微一过火，就从"吸睛"变成"标题党 / 荐股 / 造谣"**。
> 本文件收录 60+ 组 ✅ 可以写 vs ❌ 不能写的对照，写稿前必看。
>
> 使用方法：写完稿后，对照本表自检；任何一句落在 ❌ 栏的表述都必须重写。

---

## 一、标题层反例（最高发区）

### 1.1 震惊体（头条算法直接限流）

| ❌ 不能写 | ✅ 可以写 | 关键差异 |
|---|---|---|
| 震惊！A 股惊天反转，99% 散户不知道！ | 6000 点买的银行股，3400 点创新高——打脸所有空头 | 用**具体数字和反差事实**代替"震惊！" |
| 看到最后我惊了！这只股票太猛！ | 17 连板！4700 倍市盈率：散户接盘是福是祸？ | 用**具体连板数 + 市盈率**代替虚悬念 |
| 千万别错过！这波暴涨刚开始！ | 稀土暴涨 44.61%！供需格局彻底逆转 | 用**已发生的数据**代替"刚开始" |
| 这一消息炸了整个市场！ | 三部门重磅新政落地，氢能商业化元年启幕 | 用**事件主体 + 核心动作**代替"炸了" |
| 太疯狂了！A 股史上最猛！ | 27 天 7 板，再 17 连板，市值飙至 220 亿 | 用**量化轨迹**代替"史上最" |

### 1.2 荐股 / 承诺收益（红线，违规即封号）

| ❌ 不能写 | ✅ 可以写 | 关键差异 |
|---|---|---|
| 锋龙股份必涨！闭眼买入翻倍在即！ | 锋龙股份的资本赌局，散户接盘是福是祸？ | 指向性建议 → **追问立场** |
| 跟着买稀土，稳赚 50%！ | 稀土暴涨 44.61%！供需格局彻底逆转 | 预测收益 → **已发生涨幅** |
| 现在冲 CPO，下一个十倍股！ | 10 倍股集中营！AI 年报全复盘：五大细分龙头梳理 | 买卖动作 → **盘点式复盘** |
| 重仓这只！年底前必翻倍！ | 这家公司一季报净利 +262%，机构重仓 | 承诺收益 → **事实陈述** |
| 无脑买入，3 个月赚一套房 | 主线清单 + 风险提示 + 观察窗口 | 无脑引导 → **附带边界** |

### 1.3 虚悬念（泄题式标题党，头条打压）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 看完我惊了！A 股发生了什么？ | 《当初骂人的、割肉的、举报封号的，如今集体沉默》 |
| 千万别点，点了就停不下来 | 《17 连板！4700 倍市盈率：锋龙股份的资本赌局》 |
| 这件事我不敢信！ | 《6000 点买的银行股，3400 点创了新高》 |
| 看到这个数字你会哭 | 《A 股暴跌 3000 点，银行股却涨 40%》 |

**判断标准**：标题要"**具体到让人不用读正文都能知道讲了什么**"，而不是"看完再说"。

### 1.4 夸张副词堆砌 / 符号污染

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 🔥🔥🔥 暴涨！！！❗️❗️❗️ | 暴涨 44.61%，板块估值业绩双升 |
| 【重磅】【突发】【速看】【必读】 | 三部门重磅新政落地 |
| 绝对！必看！史诗级！惊天！反转！ | 27 天 7 板，再 17 连板，市值飙至 220 亿 |
| ⚡⚡⚡A 股爆了！！⚡⚡⚡ | A 股放量破万亿，五大主线全面开花 |

**规则**：标题最多 1 个感叹号、1 个问号；**禁用 emoji**；禁用【】【】【】并列 tag 堆砌。

### 1.5 人身攻击 / 失实指控

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 大蓝就是骗子！这种网红必须封杀 | 健身教练出身的财经网红大蓝，因非法荐股被永久封号 |
| XX 基金经理能力拉胯，割韭菜一把好手 | XX 基金今年跑输基准 XX 个百分点，重仓股 A、B、C |
| XX 公司财务造假，老板快跑路了 | XX 公司收到交易所问询函，涉及应收账款异常增长 |
| 董事长被查了，公司完蛋 | 公司公告董事长"因个人原因"辞职，股价早盘跌 7% |

**规则**：涉及人物或公司的负面表述，必须**有公开信源 + 事实陈述**，不做性格 / 能力层面的贬损判断。

## 二、首段反例

### 2.1 套话开局（头条最不吃）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 近日，随着宏观经济形势的持续变化，A 股市场出现了新的走势特征…… | 27 天 7 板，再 17 连板，市值飙至 220 亿！4700 倍市盈率的狂欢背后，是业绩仅 3000 万的残酷现实。 |
| 众所周知，银行股一直以来被认为是防御性板块…… | 6000 点买的银行股，3000 点解套、3400 点创新高。从人人亏惨，到全场 MVP。 |
| 本文将为大家分析一下稀土板块的走势…… | 稀土年内暴涨 44.61%，供需格局彻底逆转。 |

### 2.2 交代式开局（拖慢节奏）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 今天我们来聊聊一个有意思的话题—— 10 倍股。 | 一季报披露过半，AI 算力链已跑出五大细分龙头，净利同比 +150% 到 +300%。 |
| 大家好，今天给大家带来最新的稀土板块复盘。 | 稀土暴涨 44.61%！供需格局彻底逆转。 |
| 周末又发生了很多事情，我们一件一件来看。 | 三部门重磅新政 + 海外冲击双叠加，这个周末 A 股"不平静"。 |

### 2.3 标题党式开局（与正文脱节）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 太震撼了！这件事谁都没想到！（然后正文讲的是平常的板块轮动） | 首段必须**兑现标题承诺**：如果标题说"17 连板"，首段就要给出 17 连板的数据；如果标题说"暴涨 700%"，首段就要给出 700% 的计算口径。 |

## 三、正文语言反例

### 3.1 形容词替代数字（头条死穴）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 股价近期大幅上涨 | 股价 7 个交易日内从 18 元冲到 47 元，涨幅 161% |
| 市盈率非常高 | 动态市盈率 4700 倍 |
| 业绩表现亮眼 | 一季度净利润 1.2 亿，同比 +262% |
| 机构重仓 | 公募一季度增持 2.8 亿股，仓位占比 8.7%，第一大重仓股 |

**规则**："大幅、显著、非常、较高、明显"等形容词**一律换成数字**。

### 3.2 书卷气比喻（头条读者不吃）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 犹如沧海桑田般的变迁 | 曾经"不性感"的棉毛裤，如今成了全场最靓的仔 |
| 惊涛骇浪中的一叶扁舟 | 大盘像被砸了一棍子，就它一个逆势红 |
| 春秋代序，斗转星移 | 3 年前没人看得上，3 年后都追着跑 |
| 如雷贯耳，振聋发聩 | 直接炸了盘 |

**规则**：比喻池选"**生活场景 / 街边俚语 / 流行梗**"，**不选古文成语 / 诗意化意象**。

### 3.3 温吞归因（缺乏冲突）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 各方观点不一，市场存在分歧 | **多头**说"供需逆转刚开始"；**空头**说"涨完 45% 风险都是自己的" |
| 该板块上涨有多重因素 | 一条线拧出三股劲：**出口管制**卡供给、**缅甸禁采**断海外、**新能源需求**顶住下游 |
| 业绩和估值均有所变化 | **业绩 3000 万 vs 市值 220 亿**——这个比例能讲通吗？ |

**规则**：归因要**对立化、擂台化**，不要"多重因素并置"的中立叙事。

### 3.4 开放式追问（无立场）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 这一现象值得我们深入思考 | 最后一棒，谁来接？ |
| 未来走势如何，有待观察 | 下一个翻倍王会是谁？ |
| 投资者需要理性看待 | 别让你手里的筹码，变成别人的低位筹码 |
| 该问题具有一定复杂性 | 这到底是运气，还是网红真有眼光？ |

**规则**：追问必须**有情绪、有指向、有立场**——开放式讨论是头条读者最厌烦的。

### 3.5 过度荐股化表述（哪怕没明说个股也踩线）

| ❌ 不能写（触发风控） | ✅ 可以写（合规） |
|---|---|
| 抓紧上车，窗口期不长了 | 下一个催化节点是 5 月下旬的一季报披露 |
| 重仓 + 长持 = 翻倍 | 基本面 + 催化剂 + 估值三方验证 |
| 现在不买，后悔一辈子 | 这条主线的核心逻辑还在强化 |
| 闭眼买入这 5 只 | 以下 5 家公司是该细分的代表标的（不构成投资建议） |
| 我已经满仓干了 | 机构一季度合计增持 35 亿股 |

**规则**：**不出现第一人称买卖动作、不出现"抓紧""现在""窗口"等催促性表述**。

## 四、结构层反例

### 4.1 标题 - 正文不兑现（读者最愤怒）

| ❌ 不能写 | ✅ 必须兑现 |
|---|---|
| 标题：《附 10 大核心公司名单》→ 正文只给 3 家 | 标题承诺 10 家，正文必须列出 10 家（可带所属细分、业绩亮点） |
| 标题：《散户接盘是福是祸？》→ 正文全程打太极不给立场 | 必须在文章前 1/3 给出倾向性答案（"这场狂欢多数是坟场"） |
| 标题：《一年后暴涨 700%》→ 正文只字不提 700% 怎么算出来的 | 必须给出计算口径（起点价 / 终点价 / 时间区间） |
| 标题：《5 大主线全复盘》→ 正文讲了 3 条就转入其他话题 | 5 条主线必须逐一交代 |

**原则**：**头条标题 = 合约，正文 = 履约**。标题承诺的清单 / 答案 / 数字，正文必须原封给出。

### 4.2 段落过长（移动端自杀）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 一段 8 行把所有事讲完 | 每段 ≤ 3 行；关键句独立成段 |
| 500 字一段的论证链 | 拆成 5 段，每段 1 个论点 + 1 句收口 |

### 4.3 结构平铺（无递进）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 全文都是"有观点就列一条"的并列结构 | 按"What → Why → So What → What's Next"递进 |
| 段落之间缺乏逻辑钩子 | 每段末尾一句"收口 + 下一段引子" |

### 4.4 结尾散漫（没有力度）

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 总之，这个板块值得关注。谢谢大家。 | 最后一棒，谁来接？以上内容不构成投资建议，仅供参考。 |
| 以上就是今天的分析，欢迎点赞在看关注转发收藏。 | **禁用所有互动引导**，以追问 / 清单 / 金句收束。 |
| 综上所述，我们认为，该板块存在投资机会。 | 下一个翻倍王，可能就藏在这五大细分里。以上内容不构成投资建议，仅供参考。 |

## 五、合规层反例（违规会导致限流 / 封号 / 法律风险）

### 5.1 红线词清单（头条重点打压）

| 红线类别 | ❌ 禁用词 | ✅ 替代表述 |
|---|---|---|
| 预测收益 | 必涨、稳赚、翻倍在即、包赚不赔、躺赢 | 盈利预期 / 业绩高增 / 机构普遍看多 |
| 指向建议 | 跟我买、闭眼冲、满仓干、抓紧上车、错过拍大腿 | 以下为代表标的（不构成投资建议） |
| 第一人称动作 | 我已经买了、我重仓了、我清仓了 | 机构一季度增持 / 公募仓位变化 |
| 诱导性疑问 | 还不上车？还在等什么？ | 下一个催化节点是什么？ |
| 确定性断言 | 一定会涨、必然上涨、肯定翻倍 | 存在上涨可能 / 具备反弹条件 |
| 内部消息 | 朋友透露、内部消息、老板确认 | 公开信息显示 / 公告披露 |

### 5.2 数据来源红线

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| "据说"、"有人讲"、"听说"开头的数据 | 据 [公开信源]（公告 / Wind / 交易所披露）|
| 没有出处的"某机构预测" | 据 [XX 证券 4 月 X 日研报] |
| 断章取义的单一数据（只取对自己叙事有利的） | 多维度数据并置（营收 + 净利 + 现金流） |

### 5.3 人物 / 公司表述红线

| ❌ 不能写 | ✅ 可以写 |
|---|---|
| 大蓝是骗子 | 大蓝因非法荐股被监管部门处罚、平台封号 |
| XX 公司财务造假 | XX 公司收到交易所问询函 / 监管措施决定书 |
| 董事长跑路 | 公司公告董事长辞职，暂未披露新任人选 |
| XX 股必崩 | XX 股短期涨幅已超 X%，估值处于历史 XX 分位 |

## 六、一键自检清单（写完稿贴在眼前）

发稿前 16 问，**每一项都通过才发**：

### 标题层（5 问）
- [ ] 有 ≥ 1 个具体数字？
- [ ] 有 ≥ 1 个钩子词（反转 / 冲突 / 追问 / 清单）？
- [ ] 没有出现"必涨""稳赚""闭眼冲""跟我买"？
- [ ] 没有纯震惊体（"震惊！""看完我惊了"）？
- [ ] 标题承诺的（清单 / 答案 / 立场），正文真的兑现了？

### 语言层（4 问）
- [ ] 有 ≥ 2 处生活化比喻（不是古文 / 诗意化）？
- [ ] 形容词是否全部换成了数字 / 具体事实？
- [ ] 有 ≥ 1 处 A vs B 擂台式表述？
- [ ] 有 ≥ 1 处有立场的追问（不是"值得思考"这类开放式）？

### 结构层（3 问）
- [ ] 每段 ≤ 3 行？
- [ ] 首段在前 3 句内砸出反常识 / 冲突 / 反转？
- [ ] 结尾是追问 / 清单 / 金句收束（不是公众号三件套）？

### 合规层（4 问）
- [ ] 没有第一人称买卖动作（"我已重仓""我已清仓"）？
- [ ] 负面人物 / 公司表述是否**有公开信源**？
- [ ] 所有数据都**能溯源**（公告 / 研报 / 交易所）？
- [ ] 文末附**一句话合规声明**？

## 七、"边界案例"处理参考

以下是真实创作中的灰色地带，给出头条风的默认判断：

| 场景 | 头条风处理 |
|---|---|
| 标题可以写"暴涨 700%"吗？ | ✅ 可以（已发生的客观涨幅），但禁用"还会继续暴涨"（预测）。 |
| 可以写"游资把概念炒到天上"吗？ | ✅ 可以（描述性语言），但禁用"游资联合控盘""庄家拉升"（涉嫌指控操纵市场）。 |
| 可以写"散户接盘是福是祸"吗？ | ✅ 可以（追问立场），但正文必须给出倾向性判断，不能打太极。 |
| 可以写"业绩 3000 万 vs 市值 220 亿"吗？ | ✅ 可以（事实对比），这是头条风最推荐的冲突表达。 |
| 可以写"当初骂人的如今集体沉默"吗？ | ✅ 可以（事实群像），但不针对具体个人。 |
| 可以写"健身教练出身的网红大蓝"吗？ | ✅ 可以（身份事实 + 公开信息），但禁用"不懂装懂的骗子"（价值判断）。 |
| 可以提具体股票代码吗？ | ⚠️ 谨慎：在"主线梳理 + 不构成投资建议"框架下可以列代表标的；**单独鼓吹某只股票禁用**。 |
| 可以用"韭菜""接盘侠"吗？ | ✅ 可以（头条高频词），但不要针对具体群体（如"XX 省的韭菜""女性散户"）。 |

---

> **结语**：头条风的本质是"**把真实的冲突放大到读者挪不开眼**"，而不是"**把虚假的冲突炮制成标题党**"。
> 前者是专业编辑的手艺，后者是 SEO 农场的流水线——**我们只做前者**。
