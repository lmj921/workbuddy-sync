# 头条正文范式（内容骨架级）

> 蒸馏自 12 条爆款样本的正文结构、段落拼装法、节奏控制。
> 本文件解决："知道语感和标题了，整篇头条稿怎么搭？"

---

## 一、首段 4 种破题法（选其一，情绪爆款型必用前 3 种）

### 破题法 A｜人物故事切入（情绪爆款型首选）
> "健身教练出身的财经网红大蓝，喊出'全仓猛干胜宏科技'，因非法荐股被永久封号。
> 跟风散户短期被套，骂声一片。
> 谁料半年后剧情惊天反转——该股不到一年暴涨 700%。"

**用法**：一切涉及"人物 / 主体做了某事"的反转叙事。
**结构**：`人物身份 + 高能动作` → `引发的冲突` → `反转结果`
**字数**：80–120 字，**必须在第 3 句内完成反转**。

---

### 破题法 B｜极端数据轰炸
> "27 天 7 板，再 17 连板，市值飙至 220 亿！
> 4700 倍市盈率的狂欢背后，是业绩仅 3000 万的残酷现实。
> 游资把'机器人概念'炒到天上，股价脱离地心引力。"

**用法**：极端行情、妖股、极致泡沫。
**结构**：`爆炸性数字 1+2+3` → `与之反差的基本面` → `用比喻定性（"脱离地心引力"）`
**字数**：80–150 字，**每句必须有数字**。

---

### 破题法 C｜反差对比（反常识型）
> "完整版：'6000 点买的银行股，3000 点解套、3400 点创新高。'
> A 股从 6000 点跌到 3000 点，人人亏惨，但银行股逆势涨超 40%！
> 工行、建行、农行集体创历史新高——曾经'不性感'的棉毛裤，如今成了全场最靓的仔。"

**用法**：大盘 vs 个股、预期 vs 现实、旧叙事 vs 新王者。
**结构**：`违反常识的事实` → `强化反差的数据` → `生活化比喻定性`
**字数**：100–150 字。

---

### 破题法 D｜政策 / 事件催化（主线梳理型首选）
> "三部门联合发文，氢能商业化元年正式启幕。
> 储氢、加注、系统、整车——全产业链同时起跳。
> 这一次不是主题炒作，是政策 + 订单双驱动的真实落地。"

**用法**：政策落地、行业元年、重磅事件。
**结构**：`催化事件` → `影响范围（产业链 / 板块）` → `定性（为什么这次不同）`
**字数**：80–130 字。

## 二、两大子风格的正文结构模板

### 模板 A｜情绪爆款型（行情速评 / 热点事件 / 人物反转）

```
【标题】按 title-patterns.md 钩子 1/2/4/5/6
  ↓
【首段反常识炸场】80–120 字（破题法 A/B/C）
  - 不交代"今天写啥"，直接扔冲突
  - 必须在前 3 句内砸出 ≥ 2 个数字 + 1 个反转
  ↓
【事件复盘】3–5 段，每段 2–3 行
  - 用时间线 / 节点把故事讲清
  - 每段结尾一句画面化收口
  ↓
【冲突升级段】1–2 段（核心）
  - "A vs B" 的擂台：骂人的 vs 沉默的 / 盛宴 vs 坟场
  - 把读者的情绪拉到峰值
  ↓
【为什么会这样 / 这背后意味着什么】2–3 段
  - 给 1 个核心归因（别列 1/2/3/4/5）
  - 用"少数人看见了什么 / 多数人错过了什么"的视角
  ↓
【追问式收尾】独立 1 段
  - 反问或金句（"最后一棒谁来接？""这到底是运气还是眼光？"）
  - ❌ 不加"点赞在看""关注我们"这类公众号互动引导
  ↓
【合规短声明】1 行
  - "以上内容不构成投资建议，仅供参考。"
```

**典型样本对照**：
- 网红封号反转 → 首段人物故事（A）+ 事件复盘 + 冲突升级（骂人的 vs 沉默的）+ 追问收尾
- 锋龙股份 17 连板 → 首段极端数据（B）+ 复盘连板节奏 + 冲突升级（业绩 vs 市值）+ 追问"最后一棒谁来接"

---

### 模板 B｜主线梳理型（专题 / 盘点 / 板块复盘 / 一文读懂）

```
【标题】按 title-patterns.md 钩子 3（清单承诺型）
  ↓
【首段催化炸场】80–130 字（破题法 D 为主）
  - 核心催化 + 影响范围 + 定性
  - 建立"为什么现在是这条主线"的认知
  ↓
【核心数据速递】2–3 段，高数字密度
  - 板块涨幅 / 业绩增速 / 产业链数据
  - 每段 ≥ 3 个数字
  ↓
【X 大主线 / X 大细分逐一拆解】3–5 段（核心）
  - 每条主线 100–200 字
  - 固定结构：【主线 X：XX 方向】
    - 催化：XXX
    - 代表标的：XXX（2–4 家公司，不展开推荐）
    - 业绩 / 数据亮点：XXX
    - 画面化一句（这条线为什么硬）
  ↓
【核心公司全名单】1 大段（清单兑现，主线梳理型必选）
  - 以表格或竖排列表呈现
  - 每家公司 1 行：【公司名（代码）：所属细分 + 业绩 / 技术亮点】
  - ❌ 不配投资评级，不给买卖建议
  ↓
【风险 / 边界】1 段
  - 这条主线的主要风险点（政策不及预期 / 订单兑现 / 估值过高）
  - 这一段是合规防火墙，不能省
  ↓
【展望式收尾】独立 1 段
  - 下一个催化节点 / 观察窗口
  - 一句反问（"下一个翻倍王会是谁？"）
  ↓
【合规短声明】1 行
```

**典型样本对照**：
- 10 倍股集中营 → 首段 AI 年报定调 + 五大细分主线 + 核心公司表 + 下一个翻倍王追问
- 氢能元年 → 首段三部门新政 + 4 主线（储氢 / 加注 / 系统 / 整车）+ 10 大公司表
- 稀土暴涨 → 首段 44.61% + 双杀催化 + 供需数据 + 核心标的 + 边界提示

## 三、段落 7 种基本句型（头条专用乐高）

### 句型 1｜数字三连砸
> 27 天 7 板。17 连板。市值 220 亿。4700 倍市盈率。业绩 3000 万。
> **【节奏】** 短短短长，最后一句落反差。

### 句型 2｜A vs B 擂台句
> 当初骂人的、割肉的、举报封号的，如今集体沉默。
> 业绩 3000 万 vs 市值 220 亿。
> 少数人的盛宴 vs 多数人的坟场。

### 句型 3｜反常识反转句
> A 股从 6000 点跌到 3000 点，人人亏惨——**但银行股逆势涨超 40%**。
> 曾经"不性感"的棉毛裤，如今成了**全场最靓的仔**。

### 句型 4｜生活化比喻句
> 股价脱离**地心引力**。
> 烟花再美也会灭。
> 这场资本狂欢，注定是**少数人的盛宴、多数人的坟场**。
> 高股息 + 中特估，彻底改写市场逻辑。

### 句型 5｜第二人称痛点句
> 别让**你手里的筹码**，变成别人的低位筹码。
> **你错过的**不是一轮行情，是一个时代。

### 句型 6｜追问落幕句
> 最后一棒谁来接？
> 这到底是运气，还是网红真有眼光？
> 散户接盘是福是祸？
> 谁是下一个翻倍王？

### 句型 7｜清单速递句（主线梳理专用）
```
▍主线一：CPO —— AI 算力的光通道
  催化：英伟达 GB300 量产 + 800G 放量
  标的：中际旭创、新易盛、天孚通信
  亮点：一季度净利同比 +262%
  一句定调：卖水人逻辑最硬的一条线。

▍主线二：存储 —— 涨价周期共振
  ...
```

## 四、小标题 8 种套路

| 类型 | 示例 | 适用子风格 |
|---|---|---|
| 冲突型 | "少数人的盛宴，多数人的坟场" | A |
| 反转型 | "封号之后，剧情才刚开始" | A |
| 数字型 | "17 连板是怎么炒出来的？" | A |
| 主线型 | "▍主线一：CPO —— AI 算力的光通道" | B |
| 设问型 | "为什么这次不一样？" | A/B |
| 清单型 | "10 大核心公司全梳理" | B |
| 警示型 | "最后一棒，谁来接？" | A |
| 展望型 | "下一个翻倍王，可能在哪？" | B |

**头条小标题 3 原则**：
1. **必须信息密度高**（"关于 XX 的分析"这类空洞标题禁用）
2. **最好带数字或冲突词**
3. **每个小标题之间要有"递进感"**（从 What → Why → So What → What's Next）

## 五、结尾 3 种模式（**禁用公众号三件套**）

### 模式 1｜追问金句收（情绪爆款型首选）
> "最后一棒，谁来接？
> 这场资本狂欢，注定是少数人的盛宴、多数人的坟场。
>
> 以上内容不构成投资建议，仅供参考。"

### 模式 2｜清单展望收（主线梳理型首选）
> "下一个翻倍王，可能就藏在这五大细分里。
> 下一个关键窗口：五月下旬的一季报披露 + 英伟达 GTC 春季峰会。
>
> 以上内容不构成投资建议，仅供参考。"

### 模式 3｜反常识定性收（反常识颠覆型）
> "棉毛裤成了全场最靓的仔——改写的不只是银行股的估值，还有 A 股十年的审美。
>
> 以上内容不构成投资建议，仅供参考。"

### 🚫 头条结尾严禁

- ❌ "点个在看，股票大赚"（公众号的，头条不吃）
- ❌ "关注我们，每日复盘"（头条有独立关注机制，别照搬）
- ❌ "觉得有用点个赞 👍"（emoji + 乞求式互动，限流）
- ❌ 长篇合规声明（头条短版一行足矣，长版会让读者出戏）

## 六、配图规则（头条信息流强依赖封面）

| 位置 | 建议内容 |
|---|---|
| 封面图（最重要） | 主题大字 + 冲突感画面（K 线红绿对比 / 人物特写 / 数据柱状图）。头条信息流分发严重依赖封面点击率。 |
| 首段后 | 核心数据截图（涨幅榜 / 市盈率对比 / 业绩表） |
| 每条主线段落 | 板块 K 线 / 公司 Logo / 产业链图 |
| 冲突升级段 | 强视觉对比图（涨幅 vs 业绩 / 市值 vs 利润） |
| 结尾 | 无需配图，短声明即可 |

**封面选图 3 原则**：
1. **一眼看懂主题**（图上最好有大字数字："17 连板""4700 倍"）
2. **有冲突感**（红绿对比 / 箭头暴涨 / 人物表情）
3. **禁用模糊版权图**（头条对搬运图打压严）

## 七、一键改写 Prompt 骨架（供内部调用）

```
把下面这篇稿件改写为头条吸睛风（子风格：[情绪爆款型 A / 主线梳理型 B]）：

[原稿]

要求：
1. 标题按 _channels/toutiao/title-patterns.md 的 6 种钩子任选：
   - A 型优先用钩子 1（反转打脸）/ 钩子 2（数据冲击）/ 钩子 6（悬念追问）
   - B 型优先用钩子 3（清单承诺）
   - 标题必须含 ≥ 1 个数字 + ≥ 1 个钩子词
2. 首段按对应破题法（A 型选 A/B/C 破题法，B 型选 D 破题法）
3. 正文按对应模板结构（模板 A / 模板 B）
4. 每段 ≤ 3 行；关键数据 / 结论独立成段
5. 必须包含：
   - ≥ 2 处生活化比喻（棉毛裤 / 地心引力 / 烟花 / 坟场等）
   - ≥ 1 处 A vs B 擂台句
   - ≥ 1 处第二人称痛点句（可选，A 型推荐）
6. 结尾按 3 种模式之一，**禁用公众号互动三件套**
7. 全文合规自检：对照 antipatterns.md，不踩反例红线
8. 字数控制：A 型 1000–1500 字，B 型 1500–2500 字

输出格式：
- 建议标题（3 选 1，至少 1 个钩子不同）：
- 正文（按模板结构）：
- 自检清单：
  □ 标题数字 ≥ 1 个
  □ 标题有钩子词
  □ 首段 3 句内有反常识 / 冲突
  □ 全文有 ≥ 2 处生活化比喻
  □ 没有"必涨""稳赚""跟我买"等违规词
  □ 主线梳理型：标题承诺的清单在正文兑现
  □ 结尾没有公众号式互动引导
```
