# 公众号正文范式（内容骨架级）

> 蒸馏自样本的正文结构与段落拼装法。
> 本文件解决："知道语感了，怎么把整篇稿子搭起来？"

---

## 一、导语 3 种破题法（选其一）

### 破题法 A｜情绪直给
> "今天A股的表现，那叫一个坚韧。"
> "没有八卦，不知道怎么写复盘！漫山遍野的红！"

**用法**：日报 / 复盘 / 热点评论。1 句话定情绪基调。

### 破题法 B｜场景切入
> "最近养殖板块的热度突然回来了。"
> "外面霍尔木兹海峡又堵上了，全球市场一阵颠簸，A股看了看，转头搞自己的事。"

**用法**：解读稿 / 深度稿。用一个"你能想象的场景"把读者拉进来。

### 破题法 C｜悬念提问
> "养殖板块到底在演哪出戏？"
> "那最近这波涨价是咋回事？"

**用法**：机制解读 / 拐点判断。问完立刻答，别吊胃口超过 1 段。

## 二、主体结构模板

### 模板 1｜每日复盘 / 盘后速评（最高频）

```
【破题】1 行
  ↓
【大盘数据】2–3 段
  - 三大指数涨跌幅
  - 成交额、涨跌家数
  - 今日最大特征（放量/缩量/分化/抱团）
  ↓
【主线盘点】3–5 段，每段一条主线
  - 主线 A：板块表现 + 上涨原因 + 一句画面化收口
  - 主线 B：同上
  - 主线 C（选配）：拖后腿的板块
  ↓
【一句话总结】独立成段，加粗
  ↓
【盘后大事 1/2/3】编号列表
  - 每条 2–4 行：事件 + 一句解读
  ↓
【结尾三件套】
  - 观察窗口（下一步看什么）
  - 互动引导
  - 合规声明
```

### 模板 2｜板块/个股事件解读

```
【破题】1 行（反差/情绪/悬念）
  ↓
【事实陈述】2–3 段
  - 发生了什么（时间/主体/数据）
  - 市场第一反应（涨跌/异动）
  ↓
【为什么发生】2–3 段（串式归因）
  - 催化剂 1 + 催化剂 2 + 催化剂 3 = 结果
  - 每段后一句"人话翻译"
  ↓
【这意味着什么】2 段
  - 对行业 / 对产业链 / 对普通投资者
  - 多方观点并置（至少 1 家看多+1 家看空）
  ↓
【现在到了哪一步】1 段
  - 画面化类比（黎明前最黑 / 钝刀子磨人）
  ↓
【结尾三件套】
```

### 模板 3｜周度 / 一周研判

```
【主题金句】开篇
  ↓
【一句话定调】本周主旋律
  ↓
【5–7 个子板块】各自 100–300 字
  - 子板块：核心事件 + 数据 + 机构观点 + 一句展望
  ↓
【下周焦点】3–5 个事件清单
  ↓
【数据图表区】
  - 指数表现
  - 主力资金
  - 行业景气
  ↓
【结尾三件套】
```

## 三、段落 6 种基本句型（乐高积木）

### 句型 1｜事实+翻译
> 营收同比涨了 26%，看着不错；但净利润从 4800 万骤降到 866 万，跌了 82%。
> **【翻译】** 钱没赚到，先被业绩打了脸。

### 句型 2｜因果链
> 养殖户一听 → 变成集体惜售 → 集团猪企主动缩量 → 资金进场抄底 → 五一备货 → 猪价弹上来。

### 句型 3｜对比
> 散户亏三个月就清栏了，大企业能扛一年。

### 句型 4｜金句收口
> 一句话总结今天：放量、分化、热点极度集中。
> 过去是快刀割肉，现在是钝刀子磨人。

### 句型 5｜画面化
> 有点像黎明前最黑的那段时间，天还没亮，但东边已经隐约有点光了。
> 外面的风浪 A 股管不了，但内需的油门踩多深，自己能做主。

### 句型 6｜编号速递
```
1、[事件] — [一句解读]
2、[事件] — [一句解读]
```

## 五、结尾"三件套"模板

### 观察窗口
> "下一轮谈判能不能谈拢，现在没人知道。局势一旦再起波澜，能源价格是最直接的受益方向。"
> "等方向清晰了，等量能起来了，再动，比今天硬冲要稳得多。"
> "磨底期越煎熬，拐点到来时的弹性往往越大。"

### 互动引导（公众号专属）
> "如果您想看每日复盘、券商金股、一周策略、资金流向，请点下面的名片加关注 👇"
> "点个'分享'，股票大涨"
> "点个'在看'，股票大赚~"
> "欢迎分享、点赞、在看"

### 合规声明（公众号长版）
```
投资有风险，入市需谨慎，文中观点不构成投资建议，仅供参考。
文中数据内容及数据来源于网络、公开研报及数据库，如有侵权可删除。
风险提示：以上内容为使用 [数据工具] 进行智能分析之样例，不构成投资建议。
```

**短版**（个人 IP 派）：
> 本文不构成投资建议，你们风险自负。

## 六、配图位建议（公众号必排图）

| 位置 | 建议内容 |
|---|---|
| 头图 | 主题大字 + 品牌水印（1.91:1 或 1:1） |
| 首屏后 | 市场数据截图 / 指数分时图 |
| 每个主线段落末尾 | 板块涨幅图 / 个股 K 线 |
| 盘后大事区 | 相关事件配图（政策截图/产品图） |
| 结尾 | 引导名片 + 二维码 |

> 移动端每 3–5 段插 1 张图，避免"文字墙"。

## 七、一键改写 Prompt 骨架（供内部调用）

```
把下面这篇严肃风稿件改写为公众号"[自选哥派/推手派/猫笔刀派]"风格：

[原稿]

要求：
1. 标题按 _channels/wechat/title-patterns.md 的 7 种钩子任选
2. 导言换为 3 种破题法之一，去掉所有"近日""随着"
3. 每段不超过 3 行，重点句独立成段
4. 每个数据段后加一句"人话翻译"
5. 归因改为串式因果链（别列 1/2/3）
6. 加结尾三件套：观察窗口 + 互动引导 + 合规声明
7. 严禁：具体个股+买卖建议、承诺收益、震惊党

输出格式：
- 建议标题（3 选 1）：
- 正文：
- 自检结论（对照 _common/quality-rubric.md 7+1 条）：
```
