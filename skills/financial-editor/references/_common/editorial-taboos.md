# 编辑禁忌（病态句式 & 高频错区）

> 比合规层低一级的"职业禁区"。命中不停稿，但标准稿件不能出现。

## 一、7 种病态句式（详见 title-rules.md，此处扩到正文）
1. 无主句 / 无谓语
2. 长定语连环套："一个在近期引发广泛关注的由多部门联合发布的关于 XX 的意见"
3. 被动滥用："该政策被市场广泛认为是……"（改主动：市场普遍认为该政策……）
4. 语序倒装且无必要："地震的发生是昨天晚上在 XX"
5. "对/对于/关于"开头的无谓句："对于 XX 问题，各方看法不一"
6. 重复表述："亲眼目睹""免费赠送""提前预告""最终结果"
7. 程度词滥用："十分非常极其特别极度" —— 最多留 1 个

## 二、错别字/易混词高发区

| 正 | 误 | 备注 |
|---|---|---|
| 部分 | 部份 | 统一用"部分" |
| 账户 | 帐户 | 财务类统一"账" |
| 做大做强 | 作大作强 | — |
| 截至 | 截止 | "截至"=到某时点；"截止"=动作完成 |
| 蝉联 | 缠联 | — |
| 融资 | 溶资 | — |
| 参股 | 参股 ≠ 控股 | 不可混用 |
| 拟 | 已 | "拟"=计划，"已"=完成，差天差地 |
| 同比 / 环比 | 不可混用 | 必须明确标注 |
| 市值 / 估值 | 不可混用 | 市值=股价×总股本；估值=模型输出 |

## 三、单位与数字

- **元** 与 **万元** 与 **亿元**：不得混用，稿内统一
- **百分比** vs **百分点**：利率从 3% 升到 4%，是"上升 1 个百分点"不是"1%"
- **bp**：1bp = 0.01%。严肃风首次出现需解释
- **万亿** vs **千亿**：财报里最容易出错，必须复核
- **股** vs **份**：基金用"份"，股票用"股"
- **港元 / 人民币 / 美元**：跨市场数据必须标币种

## 四、引用错区

- **研报**：必须"机构 + 研究员 + 日期"，不能只写"某券商研报"
- **官员表态**：必须"会议/场合 + 职务 + 原文"
- **公司公告**：必须"公告日 + 公告类别"，不得用"据悉"代替
- **外媒**：引用须标注具体媒体（路透/彭博/FT），不写"外媒"

## 五、配图禁忌（详见 image-rules.md）

- 领导人照片必须用官方授权图库
- 不用尸体、血腥、灾难现场特写
- 不用竞品 logo、水印他家媒体图
- K 线截图必须带清晰来源水印

## 六、结构禁忌

- 通篇无小标题（超过 800 字必须分段）
- 所有段落同长度（缺节奏）
- 结尾戛然而止（必须有"趋势展望/下一步观察"收口）
- 整篇无一个数字（财经稿无数 = 没说服力）
