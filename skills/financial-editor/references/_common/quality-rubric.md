# 稿件质量自检表（Quality Rubric）

> 出稿前必过闸。本表是"资深财经编辑"的最小合格线：**7 条基础项必须全部 ✅，1 条加分项尽量 ✅**。
> 任意基础项未达标，稿件禁止交付。

交付格式：每条后附 `✅ / ⚠️ / ❌` + 一句话理由，跟稿附上。

---

## 基础项（7 条，缺一不可）

### 1. 标题点明主旨，角度多元且鲜明，对用户有吸引力
- **主旨**：读完标题就知道文章讲了什么事、什么结论/数据/趋势。
- **角度**：不止复述事实，要挑出"对用户最有信息价值"的切口（谁受益/谁受损/信号意义/反常处）。
- **吸引力**：数字优先、冲突优先、反常优先；拒绝"某某某发布某某某"这种无信息量标题。
- **参考**：`title-rules.md`、`_channels/<channel>/title-patterns.md`。

### 2. 数据 / 行情变化捕捉及时、准确，能把握变化背后的信号
- **可追溯**（硬标准）：稿中每一个硬数字都能追溯到 `workflows/data-first.md` 的实际检索结果。凭记忆 / 估算 / 未注明来源 = ❌ 直接打回。
- **及时**：行情稿时间戳精确到分钟；突发快讯 T+5 分钟内动笔；行情读数最近 15 分钟内，事件事实最近 6 小时内。
- **准确**：数据与来源一一对齐，百分比、同环比、基数口径清晰标注。
- **信号**：不止列数据，要回答"这个变化意味着什么"——拐点？背离？异动？预期差？
- **反面**：只罗列"涨了跌了"，没有信号解读 = ❌；数字写了但没有时间戳和来源 = ❌。

### 3. 有归因，且归因维度多元
单一归因（如"都是美联储的锅"）一律 ⚠️。至少从 **2 个维度**交叉解释：
- 宏观（货币/财政/地缘）
- 中观（行业景气/产业链/政策）
- 微观（公司基本面/情绪/资金）
- 技术面（量价/筹码/衍生品）
- 外部事件（海外市场/突发）

**归因句式示例**：
"本次下跌是三重因素叠加：① 美债收益率上行挤压估值（宏观）；② 北向资金连续 3 日净流出（资金面）；③ 权重板块业绩不及预期（基本面）。"

### 4. 归因与趋势解读逻辑清晰、观点鲜明且客观
- **逻辑清晰**：先事实 → 再归因 → 再推论，顺序不能乱。
- **观点鲜明**：有判断，不和稀泥，不堆砌"专家认为/业内人士称"而无立场。
- **客观**：不用情绪化词汇（"惨烈""暴击""血洗"在严肃风属违规），数据和引述分清。
- **自检话术**：读完一段，能否一句话复述你的观点？能 = ✅，不能 = ⚠️。

### 5. 分析解读重在未来走势，且多渠道观点归纳
- **向前看**：稿件不能停在"发生了什么"，必须说清"接下来看什么"（关键变量、观察窗口、触发条件）。
- **多渠道**：至少引用 **2 个以上来源**（卖方研报 / 公募基金 / 监管表态 / 海外机构 / 产业方），避免单一信源。
- **结构建议**：短期（1 周内）/ 中期（1 季度内）/ 长期（半年以上）分层给出。

### 6. 平衡多方观点，避免推荐个股嫌疑
- **平衡**：看多必附看空声音，反之亦然，给出双方**依据**而非只贴结论。
- **个股红线（严）**：
  - ❌ "XX 股票值得买入"
  - ❌ "建议配置 XX、XX、XX"
  - ✅ "XX 所在板块受益程度较大，其估值已修复至 ..."（说事实不给操作建议）
- **合规兜底**：稿件末尾必带"本文不构成投资建议"。

### 7. 用语简洁、不啰嗦，避免专业术语堆砌，通俗易懂
- **简洁**：一句话讲清的不用两句；去虚词（"的""了""其实""实际上""我们可以看到"）。
- **去术语**：首次出现的术语必须白话解释一次（"LPR（贷款市场报价利率，银行放贷定价的基准）"）。
- **文风**：让一个读过大学但不做金融的人，能一遍读懂。
- **反面**：整段都是"利差""久期""贴现率""β系数"而无人话翻译 = ❌。

## 加分项（1 条，满足即加分）

### ★ 透传独特的信息源或数据源
稿件中出现下列任一，视为加分：
- **自有数据**：编辑部独家统计、爬虫测算、问卷调研结果
- **独家信源**：一手采访、非公开渠道（合规前提下）、现场细节
- **稀缺视角**：冷门但关键的数据维度（如从海关进出口反推产业链景气度）
- **跨市场联动**：A 股 + 港股 + 美股 + 商品 + 汇率的交叉视角

> 有加分项的稿件，在同等合规与质量下，具备显著的传播与专业区隔优势。
