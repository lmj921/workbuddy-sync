# 模板：深度解读（Deep Read）

**适用场景**：explainer 回答不完的复杂议题、趋势研判、产业链深挖、独家调查。
**字数**：3000–8000 字。
**SLA**：重大事件 6–24 小时内。

---

## 骨架

```
【标题】主标：结论 / 副标：论据骨架 或 主标：现象 / 副标：深度拆解

【引子｜场景化开篇】（150–300 字）
- 一个具体的场景/人物/数据反差，把读者拉进议题
- 不要"近日，XX 备受关注"式空话

【核心论点】（1–2 句明确亮出观点）
本文的核心判断是：[结论]。论据有 3 条：[论据 1]、[论据 2]、[论据 3]。

【论据 1｜标题】
- 事实陈述
- 数据支撑（图/表）
- 归因分析
- 中间小结

【论据 2｜标题】
（同上结构）

【论据 3｜标题】
（同上结构）

【反方声音】（必设）
- 谁在反对我们的结论？
- 他们的依据是什么？
- 我们为什么仍然坚持原判断？（或开放地承认不确定性）

【前瞻判断】
- 短期 / 中期 / 长期分层
- 关键变量 / 触发条件 / 观察窗口

【给谁看的启示】（按读者分层）
- 对宏观研究者
- 对行业从业者
- 对普通投资者

【数据来源与方法论说明】
- 本文数据来源、统计口径、时间窗口
- 若为原创测算，需公开测算方法

【合规声明】
```

## 深度稿的"厚度"来源

1. **独家数据**：自有统计/爬虫/调研（加分项，见 quality-rubric ★）
2. **一手采访**：至少 2 位专家 / 产业内人士 / 监管人士
3. **交叉视角**：A股 + 港股 + 美股 + 商品 + 汇率的联动
4. **历史比较**：至少一次"当下 vs 历史类似时点"的对标
5. **国际视角**：海外类似案例 / 海外机构观点

## 必避雷区

- ❌ 堆砌研报观点无自己判断
- ❌ 全文"有人说/有人认为"，没点名机构
- ❌ 开头讲了 2000 字历史背景才进入正题
- ❌ 只看多或只看空，没给对立观点
- ❌ 结尾"让时间给出答案"（懒惰收口）

## 开篇 5 种钩子

1. **反差钩**：A 板块涨 30% 的同时，B 板块跌 15%——同一政策下为何冰火两重天？
2. **人物钩**：4 月 22 日下午 3 点，某百亿私募基金经理盯着屏幕……
3. **数据钩**：一个让所有分析师集体上调预测的数字：X.XX%
4. **历史钩**：上一次我们看到这个组合，是 2015 年 6 月——那次的剧本是……
5. **悖论钩**：经济数据强劲，股市却连续下跌，市场到底在定价什么？

## 结尾收口范式

- **风险点式**：接下来，X、Y、Z 三件事会给判断证伪或证真
- **节点式**：下一个关键窗口是 [具体日期/事件]
- **分层式**：对不同参与方而言，当下应该做的事/应该关注的是……（不给买卖建议）
