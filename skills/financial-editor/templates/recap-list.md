# 模板：盘点/榜单（Recap List）

**适用场景**：周度 / 月度 / 季度 / 年度总结，热点事件复盘，行业榜单。
**字数**：1500–4000 字。
**形态**：结构化清单 + 小结 + 趋势判断。

---

## 骨架

```
【标题】时间周期 + 主题 + 数字
示例：2025 一季度 A 股十大关键事件
示例：4 月外资重仓股 TOP20：白酒回归，半导体加仓

【开篇导语】（100–200 字）
- 时间窗口
- 本期盘点依据（数据口径 / 评选方法）
- 一句话提炼本期主旋律

【TOP N 榜单 / 大事记】
──────────────
01. [事件 / 主体名称] ｜ [量化指标]
   - 一句话摘要
   - 关键数字
   - 意义 / 启示

02. [...]
   - ...

【横向对比】
- 与上一周期（上月 / 上季 / 上年）对比
- 排名变化、新晋、退出

【趋势小结】
- 从榜单中提炼 2–3 个结构性趋势
- 哪些板块在崛起？哪些在退潮？

【下期看什么】
- 下一周期的关键变量
- 值得持续跟踪的主体

【数据来源与评选方法】
- 数据口径、时间窗口、评选标准
【合规声明】
```

## 盘点稿的"信息密度"公式

**信息密度 = 数字数量 × 对比维度 × 异动发现**

- **数字**：每条榜单至少 2 个数字（量化指标 + 变化幅度）
- **对比**：至少一个时间维度的对比（上期 vs 本期）
- **异动**：主动挑"新晋黑马 / 意外滑落 / 超预期 / 不及预期"的条目展开讲

## 榜单稿三问自检

1. 删掉排名数字，榜单依然有可读性吗？（应该依然有）
2. 读者能从榜单中感知到**趋势**，还是只看到一堆名字？
3. 有无至少 1 个"意料之外"的条目被点出？

## 盘点稿标题范式

- `[周期] [主题] [TOP N]`：2025 一季度 A 股分红榜 TOP20
- `N 大 [主题] 盘点`：4 月十大政策事件盘点
- `[周期] 回顾：[核心趋势]`：4 月回顾：从"抱团消费"到"布局 AI"
- `N 张图看 [周期]`：5 张图看懂 4 月外资持仓变化

## 禁区

- ❌ 只列不评（纯名单 = 没生产编辑价值）
- ❌ 没有数据口径说明（"TOP20 按什么排序？"读者问不出答案）
- ❌ 推荐型榜单："最值得买的 10 只股票" —— 触及个股红线
- ✅ 客观型榜单："一季度机构调研次数 TOP10" —— 事实导向可发
