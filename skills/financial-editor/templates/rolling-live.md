# 模板：滚动播报（Rolling Live）

**适用场景**：持续 1–72 小时的重大事件（议息会议、财报季、重大会议、突发危机）。
**形态**：时间倒序 Timeline + 关键节点加亮。

---

## 骨架

```
【专题标题】
【开篇导语】（2–3 行）
- 事件是什么、为何滚动报道、关注哪些关键节点

【Timeline｜时间倒序】
──────────────
🕐 最新时间节点｜[主体][动作][量化]
- 一句话事实
- 一句话影响/数据

🕐 N-1 时间节点｜[...]
...

【当前焦点】（置顶区，随事件更新）
- 下一个关键时点：
- 市场主流预期：
- 需要观察的信号：

【相关解读】
- 📌 关联 explainer / deep-read 文章链接
```

## 更新节奏

| 事件类型 | 更新频率 | 持续时长 |
|---|---|---|
| 议息会议 | 30 分钟 | 12 小时 |
| 财报季重点公司 | 1 小时 | 24 小时 |
| 地缘突发 | 15–30 分钟 | 持续至事件平息 |
| 国新办发布会 | 实时 | 会议结束后 2 小时 |

## 写作要点

1. 每个节点独立成条，可单独阅读
2. 时间戳精确到分钟
3. 关键节点（决议公布/高管发言/行情异动）**加粗**
4. 节点之间不写过渡句，保持时间轴干净
5. 置顶区"当前焦点"每次更新必须覆盖刷新

## 节点写作模板

```
🕐 14:32｜证监会新闻发布会：严打财务造假，年内立案 42 起
- 证监会稽查局局长表示，将建立上市公司"ST+立案"联动机制
- A 股反应：金融股直线拉升，中信证券涨 2.3%（数据来源：Wind，14:35）
```

## 收官节点（滚动结束）

滚动结束时必发一条"**全景回顾**"：
- 事件全过程 5 个关键节点压缩版
- 市场整体表现（主要指数+重点板块+汇率商品）
- 延伸阅读（关联 explainer / deep-read）
