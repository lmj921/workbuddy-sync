# 李录 · 投资思维模型与决策框架

---

## 一、价值投资六条核心原则

李录在2024年北大演讲中系统总结了价值投资的六条核心理念，前五条来自前人（格雷厄姆、巴菲特、芒格），第六条是他本人的独立贡献：

### 第一条：股票是公司所有权的法定证明（格雷厄姆）
- 股票不仅是一张可交易的纸，它代表了公司的一部分所有权
- 在经济持续增长的过程中，股权能够保障你的购买力

### 第二条：市场先生提供的是服务，而非指导（格雷厄姆）
- 市场由个体组成，人的本性追求短期获利，往往把股票当作短期交易筹码
- "市场先生"是一个非常神经质的人，他的作用不是告诉你真正的价值，而只是提供买卖价格
- 这些价格往往远低于或远高于价值——对价值投资人来说，他提供的是服务，而不是指导

### 第三条：投资必须有充足的安全边际（格雷厄姆）
- 未来很难预测，便宜是硬道理
- 以足够低的价格买入，留出充足的安全边际，你的投资就会更安心，也更能坚持长期持有
- "我的价值投资就是从买便宜公司开始。在投资便宜公司的过程中，逐步建立起自己的能力圈"

### 第四条：投资人要明确自己的能力圈（巴菲特/芒格）
- 优秀公司能够长期持续地超越竞争对手，具有长期高于行业平均的资本回报率
- 挑选和理解这些公司并不容易，所以要建立自己的能力圈
- 知道自己哪些懂，哪些不懂——**知道能力圈的边界比知道面积更重要**
- 只投资能力圈范围内那些自己能理解的优质公司，并长期持有

### 第五条：在有鱼的地方钓鱼（芒格）
**芒格钓鱼的故事**：芒格每年夏天在明尼苏达星岛度假。岛旁就有大湖，但每次钓鱼都驱车一小时去偏远小湖。因为小湖里鱼多且竞争者少——钓鱼向导 Leroy 通过常年寻找鱼饵积累了独家知识，知道哪个湖里有鱼。每次去那些小湖，都收获满满，而且一整天只有他们一群人。

**投资类比**：
- 不需要在最大的湖里钓鱼（不需要了解所有公司和所有宏观参数）
- 关键是找到"鱼多且人不多"的湖（竞争不充分+你深刻理解的领域）
- 竞争不充分是错误定价的重要原因
- 两条法则：第一条，在有鱼的地方钓鱼；第二条，千万别忘了第一条

### 第六条：财富是经济体中的购买力占比（李录独创）
- 财富的本质不是静态的绝对数值（土地、现金、房产），而是你在经济体中所占有的**购买力比例**
- 现代经济持续复利增长，静态财富必然缩水（当年的万元户≠今天的富人）
- 价值投资的目标：在最具活力的经济体中，持有最具活力的公司的股份，从而保持和增长你的购买力占比
- 对个人投资者：在你需要消费的经济体中保持购买力
- 对全球投资者：在全球范围内选择最有活力的经济体和公司

## 二、文明三阶段框架

李录在《文明、现代化、价值投资与中国》中提出：

| 阶段 | 名称 | 核心特征 | 财富基础 | 增长模式 |
|------|------|----------|----------|----------|
| 1.0 | 狩猎采集文明 | 随气候迁徙 | 自然资源 | 无增长 |
| 2.0 | 农业文明 | 定居+农耕+帝国 | 土地+人口 | 受马尔萨斯陷阱限制，周期性兴衰 |
| 2.5 | 中间盘整期 | 工业起飞后的转型 | 各种观念冲突 | 关键关口：能否跨越中等收入陷阱 |
| 3.0 | 现代科技文明 | 自由市场+现代科技 | 市场规模+要素流通 | 自生的、复利性的、持续的增长 |

**核心洞察**：
- 现代化的本质 = 现代科技 × 自由市场经济 → 持续的复利增长
- 经济复利增长很快，但人性不变、社会治理缓慢——这种落差是一切"时代困惑"的根源
- 中国目前处在2.5阶段，面临所有国家工业起飞后都经历过的挑战
- 土地不再是3.0时代的核心财富（英国贵族的城堡从财富变成负担）
- 虚拟经济vs实体经济是伪概念（英伟达市值超过德国+意大利全部上市公司）
- 全球所有经历过工业起飞的国家，追赶阶段都只需三四十年

## 三、能力圈建设方法论

### 能力圈的本质
- **核心定义**：知道自己知道什么，更重要的是知道自己**不知道什么**
- **边界比面积更重要**：不是懂多少行业，而是清楚知道哪里是你的边界
- **诚实是基础**：知之则知之，不知则不知，绝不不懂装懂
- **只有触摸到边界，才能确认圈是否存在**：比亚迪每次跌50%以上，都是对能力圈真实性的考验

### 能力圈的层次
1. **简单了解一个行业** ≠ 能力圈
2. **能判断其未来10-20年的竞争格局** = 开始接近能力圈
3. **股价跌50%-80%时依然坚定持有** = 能力圈经受住了考验

### 能力圈的建设路径
1. 从买便宜的公司开始——价格足够便宜让你安心持有
2. 持有期间深入研究企业和生意——"买完之后必须去深入研究"
3. 从理解便宜公司过渡到理解优质且便宜的公司
4. 持续学习，复利式增长——芒格99岁还在发现新的投资标的
5. 不必着急——"长期持有的前提是真正理解，而不是为了长期持有而长期持有"

## 四、"好球"理论（棒球击球区模型）

来自《穷查理宝典》精装版第61页，李录的投资哲学核心表述：

**类比**：泰德·威廉姆斯是过去七十年来唯一单赛季打出.400打击率的球员。他将击打区划分为77个格子，只在球落入"最佳"格子时才挥棒。

**投资对应**：
- 证券投资者应持续观察各种企业的证券价格
- **大多数时候什么也不做，只是观察**
- 等待那个"速度很慢、线路又直、正好落在最爱格子中间"的好球
- 然后**全力出击**

**两大常见错误**：
1. **挥棒太过频繁**——个人和专业投资者的共同问题
2. **发现好球却无法用全部资本出击**——与前者同样有害

## 五、卖出四原则

李录很少卖出，但给出了四种卖出理由：

| 情形 | 说明 |
|------|------|
| 发现犯了错 | 第一时间卖出，不犹豫 |
| 发现更优标的 | 风险回报比（risk-reward）和潜在损失收益（downside-upside）更优时替换 |
| 极端泡沫 | 市场出现极端高估——但估值是时间维度上的概念，长期增长能力会改变"高估"的判断 |
| 迫不得已 | 作为受托人遇到赎回要求（不借债原则下的被动卖出） |

**核心态度**：
- 真正的"圣杯"公司凤毛麟角——一旦找到并理解，不要轻易丢掉筹码
- "如果因为觉得高估了卖掉它，再想买回来，你会发觉还是面临同样的问题"
- 短期高估与长期增长相比，往往没那么重要
- 人类共同的缺点：放大短期因素，缩小或忽视长期因素

## 六、投资比亚迪案例深度解析

这是中国价值投资史上最具代表性的超长期持仓案例：

### 时间线

| 时间 | 事件 |
|------|------|
| 2000年 | 从技术杂志上注意到比亚迪——全球能做可充电锂电池的只有几家日本公司，一家中国公司引起高度关注 |
| 2002年秋 | 在芝加哥与王传福第一次见面，"一聊就是好几个小时，一见如故"。随后买入第一批股份（仓位约10%） |
| 2003年 | 比亚迪收购秦川汽车进军汽车业，市场大跌。李录"没什么特别反应"——用第一性原理思考的人，商业没有边界 |
| 2007年12月 | 比亚迪电子分拆上市，募资67亿港币，度过2008年金融危机 |
| 2008年9月 | 引荐巴菲特以8港元/股入股比亚迪——芒格一生只拍过两次桌子让巴菲特"买买买"，比亚迪是其中之一 |
| 2009年 | 股价从8涨到80港元，又跌去80%以上——"我们既没有卖掉一股，也没有买下一股" |
| 2014年 | 股价闪崩50%，李录坚定回应"一定不是我，我对公司充满信心"，随后加仓300多万股 |
| 2020年 | 疫情口罩事件：比亚迪3天出图纸7天出生产线，从0到日产1亿只口罩 |
| 至今 | 持有23年未出清，比亚迪销售额从十亿增长到近万亿人民币 |

### 核心投资逻辑
1. **极低价格买入**：投资成本大概只有一两块钱，安全边际极大
2. **创始人品质**：王传福——第一性原理思维+工程师落地能力+受托人精神+永不言败+学习复利
3. **"二级市场的风险投资"**：以公开市场价格获得了VC级别的回报
4. **陪伴成长**：不仅是资金支持，还有声誉背书（引入伯克希尔）

### 案例启示
- 价值投资人不是寄生虫，是企业成长不可或缺的伙伴
- 能力圈不是静态的——每次股价大跌都在考验你是否真的懂
- 犯了错误不可怕——"犯了错误，从中学习，面对现实，永不言败，继续往前走"
- 发现"凤毛麟角"的企业家后要"值得终身珍惜"

## 七、对人性的理解

李录对人性有独特而深刻的认知：

- **动物性/人性/神性比例**："人就是从动物演化来的。动物性占了六七分，人性占了三四分，还有半分的神性。"
- **人性无好坏之分**："It is what it is。文化进化就是让人凸显人性、扩大神性和限制动物性。"
- **理解人性是自知的开始**："理解人的本性是自知的开始，也是自我提高的开始。"
- **价值投资为何难以成为主流**：因为人性没有改变——赌性会被随时发挥出来。股票可以随时买卖，就像赌场筹码一样。
- **真正的价值投资者**可能不到市场参与者的5%。

## 八、"鸡与蛋"的经济链条思维

李录用"鸡与蛋"的比喻理解经济系统：

- 经济整体是一个环环相扣的链条：企业家精神、消费者信心、官员激励、外资信任、资本市场效率、国际关系……
- 链条上每一个节点既是"鸡"也是"蛋"——都可以"下蛋"
- 激发任何一处，都可以点燃整个链条（改革开放初期，几十个农户的包产到户就点燃了40年变革）
- 不需要从某个固定方向开始，重要的是让环境足够宽松

## 九、香港资本市场论

李录对香港金融市场有独特见解：

- 英国给中国留下的一份"礼物"——相当于荷兰给英国的金融体系礼物
- 香港具备现代资本市场的所有要素：完备制度、法律、历史传统、纠纷解决机制、信用中介、国际信任
- 资本市场提供的不仅是资金，更重要的是**信用体系**（银行无法提供）
- 信用体系把普通小储户的钱聚集成巨大投资力量，最终支持企业成长和经济发展
- 中国只有香港的资本市场具备现代金融市场体系的全部要素
- 保障香港独立性 = 当年保障深圳特区独立性——两种体制并行，最终促成改革

## 十一、现代化铁律（中等收入跨越清单）

李录用来判断一个经济体能否跨越2.5阶段的关键要素：

1. **市场经济的共识**：私人决策为主、政府从指挥型转为服务型
2. **人身和财产的基本保障**：生命安全、财产安全、程序正义
3. **健全的资本市场**：让储蓄转化为投资的信用体系
4. **经济要素充分流通**：商品、服务、资本、人才、思想无堵点
5. **个人消费占GDP比重**：核心KPI——中国目前约40%（vs 印度60%、美国>70%）
6. **储蓄有效转化**：中国储蓄率约50%但主要沉淀在国有银行，未能有效循环
7. **包容性**："市场经济不管哪种人能成功，自由很重要，提供空间很重要"

## 十二、"本分"哲学

受段永平影响的核心行事原则：

- **"本分一点思考问题"**——无论是管理者还是投资人
- **"就挣自己应该挣的钱"**——你应该挣的钱是公司内在价值被实现、被认可时的那一份
- **不借债**——只有不借债才能经得起整个组合跌50%的极端情况
- **不做自己不懂的**——"绝大部分机会都不是我的"
- **"不做"的错误比"做错"更大**——看明白了但因短期波动放弃是最大的遗憾

## 十三、复利的深层理解

李录对复利有远超一般投资人的理解深度：

### 经济复利
- 过去200年除去通胀的累积回报率约6-7%，但复利结果是**75万倍**
- 复利现象是近代文明出现后才有的——之前的规律是热力学第二定律（熵增）
- 3.0经济的本质就是持续的、复利性的、自生性的增长

### 知识复利
- "知识层面的交流甚至能产生1+1>4的功用"
- 能力是可以复利式增长的——芒格99岁仍在发现新投资
- 王传福身上最值得钦佩的就是"学习的能力，持续学习的能力"——以复利形式不断深化

### 能力复利
- "我们只要每天都能够比前一天稍稍学习多一点，我们就可以走很远很远的道路"
- "每天一点点地提高我自己，这就是人生很大的乐趣了"
