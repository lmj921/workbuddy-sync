# 巴菲特思维模型参考库

> 来源：伯克希尔股东信(1965-2025)、历年股东大会发言、芒格合作对话、公开访谈、《巴菲特致股东的信》
> 标注说明：【原话】= 巴菲特本人直接表达；【推断】= 基于其框架的合理推论

---

## 一、价值投资四大支柱

### 支柱1：能力圈（Circle of Competence）

**定义**：只投资自己真正理解的生意。不是看懂财报，是能回答「这家公司5-10年后为什么还能赚钱」。

【原话】「投资人真正需要的，是正确评估所选企业的能力。注意'所选'二字——你不需要成为每家公司的专家，甚至不需要了解很多公司，你只需要能评估在你能力范围之内的公司就行了。能力圈的大小不重要，知道它的边界在哪里才最重要。」

【原话】「如果你不能用十分钟解释清楚你为什么买这只股票，那你就不应该买它。」

**「懂」的判断标准：**
- 能用一句话向10岁小孩解释这家公司怎么赚钱
- 能解释这家公司10年后为什么还在、还会更好
- 能说出竞争对手为什么无法轻易复制它

**不懂的处理**：买标普500指数基金。

【原话】「大多数投资者，不论是机构还是个人，会发现投资股票的最佳方式是买入费率很低的指数基金。」

**局限**：能力圈框架假设「不懂就不碰」，但可能导致系统性错过新范式。巴菲特承认错过谷歌和亚马逊。

---

### 支柱2：经济护城河（Economic Moat）

**定义**：企业抵御竞争、维持长期高回报的结构性优势。好公司就像被宽阔护城河保护的城堡。

【原话】「真正伟大的企业必须有持久的'护城河'来保护出色的资本回报。资本主义的动力保证了竞争者会反复攻击任何赚取高回报的商业'城堡'。」

**四种护城河类型：**

| 类型 | 含义 | 典型案例 |
|------|------|---------|
| 无形资产 | 品牌、专利、特许经营权 | 可口可乐（品牌）、GEICO（低成本品牌） |
| 转换成本 | 用户迁移代价极高 | 苹果（iOS生态+iCloud绑定） |
| 网络效应 | 用户越多，价值越大 | Visa、美国运通 |
| 成本优势 | 规模效应或独特流程带来低成本 | BNSF铁路、GEICO |

**关键判断**：护城河在变宽还是变窄？
- 变宽 = 竞争优势在增强 → 长期持有
- 变窄 = 竞争优势在侵蚀 → 考虑退出

【原话】「最重要的是确定一个产业的竞争优势及其可持续性。获得护城河宽阔的产品或服务，才能给投资者带来丰厚回报。」

---

### 支柱3：市场先生（Mr. Market）

**来源**：格雷厄姆在《聪明的投资者》中创造的寓言。巴菲特将其发扬光大。

**核心逻辑**：市场是你的仆人，不是你的主人。它每天给你报价，你可以利用它的非理性，也可以无视它。但永远不要被它的情绪左右。

【原话】「在别人贪婪时恐惧，在别人恐惧时贪婪。」

【原话】「市场是一台投票机（短期），但它是一台称重机（长期）。」

**应用方法：**
- 建立「观察列表」：提前研究好能力圈内、有宽阔护城河的公司
- 设定「好球带」：根据估值设定具备安全边际的买入价
- 耐心等待：市场恐慌时出手，市场狂热时保持冷静

【原话】「机会不常来。天上掉金子的时候，要拿桶去接，而不是拿顶针。」

---

### 支柱4：安全边际（Margin of Safety）

**定义**：以显著低于内在价值的价格买入，为估值误差和未来不确定性预留缓冲空间。

【原话】「价格是你支付的，价值是你得到的。」

**应用方法：**
- 估算内在价值（未来现金流折现的思维方式，不是精确计算）
- 以足够大的折扣买入
- 折扣越大 = 安全边际越高 = 风险越低

【推断】巴菲特的安全边际不是一个固定比例（如30%），而是取决于确定性——越确定的公司可以接受更小的折扣，越不确定的需要更大折扣。

## 二、核心投资思维模型

### 模型1：买股票就是买公司的一部分

【原话】「我是一个好的投资者，因为我是一个好的商人；我是一个好的商人，因为我是一个好的投资者。」

**退市测试**：如果这家公司明天永远退市，你还愿意持有吗？
- 愿意 = 你在投资
- 不愿意 = 你在投机

【原话】「我们最喜欢的持股期限是……永远。」

---

### 模型2：复利雪球（Snowball）

【原话】「人生就像滚雪球，重要的是找到很湿的雪和很长的坡。」

**三层含义：**
1. 「湿的雪」= 高资本回报率的好生意
2. 「长的坡」= 足够长的时间跨度
3. 起步的雪球不需要大，但需要尽早开始

【原话】「时间是优秀企业的朋友，平庸企业的敌人。」

---

### 模型3：20个打孔位（20 Punches）

【原话】「我可以给你一张只有20个孔的打孔卡，你每做一次投资就打一个孔。打完20个孔，你这辈子就不能再投资了。在这个规则下，你会非常谨慎——而你也会因此变得富有。」

**含义：**
- 好的投资机会极其稀缺
- 集中持仓（通常只持有少数几家公司）
- 深度研究比广泛撒网重要得多
- 「多元化是无知的保护」

---

### 模型4：收费桥梁（Toll Bridge）

【推断】巴菲特偏爱的商业模式——用户要从A到B，必须经过你的桥，而且没有替代路线。

**典型案例：**
- 美国运通：商户和消费者之间的支付通道
- 穆迪/标普：债券发行必须经过评级机构
- 铁路（BNSF）：货物运输的物理通道，无法轻易复制

---

### 模型5：管理层评估三问

巴菲特评估管理层用三个标准：

| 标准 | 核心问题 |
|------|---------|
| 理性 | 资本配置是否明智？是否抵制了「帝国建设」的诱惑？ |
| 诚实 | 对股东是否坦诚？好消息坏消息是否一视同仁？ |
| 独立思考 | 是否只因为「别人在做」就跟风？ |

【原话】「我们寻找三样东西：智慧、精力和正直。如果缺少最后一样，前两样会毁了你。」

---

### 模型6：伟大公司犯错是机会

【原话】「一生中只需要几次好的决策。」

**逻辑**：
- 伟大公司暂时遇到可修复的问题 → 股价暴跌 → 巨大的安全边际
- 关键判断：问题是暂时的还是永久的？护城河有没有受损？

**经典案例：**
- 1963年美国运通「色拉油丑闻」→ 买入（品牌未受损）
- 1988年可口可乐New Coke失败 → 买入（已修正错误）
- 2008年金融危机 → 投资高盛、GE（系统重要性，不会死）
- 2016年苹果暂时低迷 → 大举买入（生态系统护城河完好）

## 三、快速判断规则

| 规则 | 核心问题 | 典型案例 |
|------|---------|---------|
| 退市测试 | 永远退市也愿意持有吗？ | 可口可乐（会）vs 投机股（不会） |
| 10年测试 | 10年后还在且更好吗？ | 苹果（会）vs 诺基亚2007（不会） |
| 一句话测试 | 10岁小孩听得懂它怎么赚钱吗？ | 喜诗糖果（能）vs 衍生品交易（不能） |
| 护城河测试 | 护城河在变宽还是变窄？ | 苹果生态（变宽）vs 报纸行业（变窄） |
| 好球带 | 价格是否在安全边际内？ | 不着急，等好球 |
| 报复心测试 | 是否因情绪而非理性在做决策？ | 收购伯克希尔（报复心 → 大错） |
| 翻转测试 | 如果反过来想会怎样？（芒格的逆向思维） | 不想破产就避免过度杠杆 |

## 四、生平重大决策节点

| 时间 | 事件 | 巴菲特的决策逻辑 |
|------|------|-----------------|
| 1941年 | 11岁买入人生第一支股票 Cities Service | 初次体验市场波动，学到耐心 |
| 1951年 | 哥伦比亚大学师从格雷厄姆 | 奠定价值投资根基 |
| 1956年 | 成立巴菲特合伙基金 | 26岁起步，起始资金10.5万美元 |
| 1962年 | 开始买入伯克希尔·哈撒韦 | 后来承认是「报复性收购」，一生最大错误之一 |
| 1967年 | 收购国民保险公司 | 获得保险浮存金——「免费杠杆」，改变伯克希尔命运 |
| 1972年 | 收购喜诗糖果（$2500万） | 芒格教他「为好公司付合理价格」，告别「捡烟蒂」 |
| 1973-74年 | 熊市中大量买入华盛顿邮报 | 极度便宜+垄断地位 |
| 1988年 | 重仓可口可乐（$10亿） | 全球最强品牌，简单的商业模式，标志性长期持仓 |
| 1993年 | 换股收购德克斯特鞋业 | 高估美国制造业竞争力，自称「可能是我做过的最差交易」 |
| 1998年 | 收购通用再保险 | 获得更多浮存金，但整合过程痛苦 |
| 2003年 | 买入中石油H股 | 以约$5亿买入，2007年以$40亿卖出 |
| 2008年 | 金融危机投资高盛/GE | 「别人恐惧时贪婪」的教科书实践 |
| 2010年 | 收购BNSF铁路（$440亿） | 对美国经济长期乐观的「all-in」赌注 |
| 2011年 | 买入IBM | 后承认是失误，不够了解科技行业竞争格局 |
| 2016年 | 大举买入苹果 | 视为「消费品公司」而非科技公司，成为最大持仓 |
| 2020年 | 疫情中清仓航空股 | 承认犯错，「世界变了」 |
| 2022-24年 | 增持日本五大商社 | 低估值+分红+多元化的商业帝国 |
| 2023年11月 | 芒格去世（享年99岁） | 失去60年合作伙伴 |
| 2024年 | 大幅减持苹果，现金储备创新高 | 「市场整体估值偏高」【推断】 |
| 2025年2月 | 致股东信高度赞扬Greg Abel | 将继任者与芒格并列 |
| 2025年5月 | 股东大会宣布年底卸任CEO | Greg Abel正式接班 |
| 2025年Q3 | 大举建仓Alphabet约1785万股 | 退休前的最后一次重大建仓 |
| 2025年11月 | 最后一封致股东信：「I'm going quiet.」 | 95岁谢幕 |

## 六、已知内在矛盾（必须如实呈现，不调和）

1. **「永远不要亏钱」vs 多次重大亏损**
   - 主张：Rule No. 1: Never lose money
   - 实际：德克斯特鞋业（亏损35亿美元）、卡夫亨氏（减值数十亿）、航空股（疫情亏损数十亿）、IBM（认错卖出）
   - 他的解释：原则是方向，不是每次都能做到

2. **「不懂科技」vs 苹果成为最大持仓**
   - 长期标签：「科技股不在我的能力圈」
   - 实际：苹果一度占伯克希尔投资组合近50%
   - 他的辩解：苹果本质是消费品公司，不是科技公司
   - 批评方：这是重新定义「能力圈」来合理化决策

3. **「长期持有」vs 大幅减持苹果**
   - 主张：「我们最喜欢的持股期限是永远」
   - 实际：2024年大幅减持苹果约50%
   - 市场猜测：估值过高 / 税收考量 / 为继任者减负

4. **「现金是垃圾」vs 持有超3000亿美元现金**
   - 长期偏好：满仓好公司
   - 2024-2025：现金储备创历史新高
   - 市场解读：找不到足够便宜的标的 / 为潜在大型收购储备

5. **「多元化是无知的保护」vs 伯克希尔本身高度多元化**
   - 主张：集中持仓少数确定性高的公司
   - 实际：伯克希尔旗下超80家全资子公司，覆盖保险、铁路、能源、零售等
   - 他的辩解：「对于了解自己在做什么的人」集中好，但伯克希尔的多元化是资本配置的结果

6. **「我只是运气好」vs 连续60年超额收益**
   - 他常自嘲运气好、生在美国、生对了时代
   - 批评方：这种谦虚是策略性的，实际上他对自己的判断极度自信

7. **「做空美国的人都会输」vs 2024-25年大量囤积现金**
   - 公开表态：永远不要做空美国
   - 行动：却在美股高位大幅减仓、囤现金
   - 两种解读：谨慎 vs 矛盾
