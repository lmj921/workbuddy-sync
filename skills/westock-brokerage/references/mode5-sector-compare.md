# 模式5：多板块对比 — 券商看好家数

## 适用场景

用户提问如"医药板块和创新药板块各有多少家券商看好"、"通过券商看好数量给板块排名"等多板块对比问题时使用此模板。

- **模式标识**：`sector-compare`

## 观点态度判定规则

与模式4一致：

| 态度 | 触发关键词 |
|-----|-----------|
| **看好** | "超配"、"强烈推荐"、"首推"、"看好"、"推荐"、"增持" |
| **中性** | "标配"、"中性"、"观望" |
| **看空** | "低配"、"回避"、"谨慎"、"减持" |

## 输出格式

### Part A — 柱状图排行

使用文本柱状图直观展示各板块被看好的家数：

```
{板块名称}        {█数量对应柱状图} X家券商
{板块名称}        {█数量对应柱状图} X家券商
```

**示例**：

```
医药            ████████████████████ 20家券商
商业航天        ██████████ 10家券商
算力芯片        ████████ 7家券商
新能源车        ██████ 5家券商
```

### Part B — 总结表述

> 综合近一周 X 家券商观点，**{板块1}** 以 X 家看好位居榜首，**{板块2}**（X 家）、**{板块3}**（X 家）紧随其后。多数券商认为 {核心共识判断}。需注意 {分歧/风险提示}。

## 内容要求

- 柱状图按看好家数降序排列
- 柱状图长度与家数成正比（每家 ≈ 1 个 █）
- 总结控制在 100–150 字
- 板块名称加粗
- 所有标点使用中文标点
